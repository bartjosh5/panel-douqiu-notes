# panel-douqiu-notes

## 简介

`panel-douqiu-notes` 是一个用于归档和发布独立 HTML 页面的仓库。  
这里存放的页面以笔记、工具、展示页等形式为主，不针对特定域名或网站。  
每个页面均为独立运行，不依赖后端服务，适合直接通过 GitHub Pages 或其他静态托管方式访问。

## 目录结构

```
panel-douqiu-notes/
├── index.html          # 仓库主页 / 导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

- 每个 HTML 页面均为独立完整的文件，内嵌样式与脚本，或引用 `assets/` 目录下的公共资源。
- 页面内容以笔记、实验、工具或信息展示为主，不包含商业推广或诱导点击内容。
- 所有页面均可在现代浏览器中直接打开运行，无需额外配置。

## 维护说明

- 本仓库以归档和整理为目的，不定期更新或补充新页面。
- 页面内容可能根据需要进行调整或修正，不保证长期不变。
- 欢迎通过 Issues 提出建议或反馈，但请注意本仓库不涉及具体网站或域名相关讨论。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意页面查看。
3. 若部署到 GitHub Pages，则仓库根目录即为站点根目录。
